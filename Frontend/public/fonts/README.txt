Place a Unicode-capable TTF font here to enable proper rendering of non-Latin characters (e.g., Thai) in generated PDFs.

Recommended fonts:
- NotoSansThai-Regular.ttf (https://fonts.google.com/noto/specimen/Noto+Sans+Thai)
- Sarabun-Regular.ttf (https://fonts.google.com/specimen/Sarabun)

After adding the file as NotoSansThai-Regular.ttf, rebuild the app. The frontend will embed this font at runtime when generating PDFs.
